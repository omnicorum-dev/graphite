Detailed Description

References into an Object3D's attribute arrays for a single polygon vertex.

Indices are 0-based and correspond to the vertices, texCoords, and normals vectors of the owning Object3D. randColor is a random per-face color assigned when loading an OBJ, used by the vertex-color draw path.
