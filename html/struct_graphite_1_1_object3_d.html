Detailed Description

A 3D mesh with associated transform and optional texture.

Vertices, normals, and texture coordinates are stored in parallel arrays; faces contains one entry per polygon, each being a list of FaceIndex values that index into those arrays.

The mesh can be rendered via Camera::drawObject() and its variants.
