Detailed Description

Graphite — a software-rasterization 2D/3D rendering library.

Graphite provides a CPU-side pixel canvas with a full suite of 2D drawing primitives (rectangles, circles, ellipses, lines, triangles, text) as well as a lightweight 3D pipeline (perspective projection, texture mapping, z-buffering, back-face culling, and diffuse shading) built on top of GLM.

Quick-start

#include <graphite.h>
Graphite::Canvas c(800, 600);
c.fill(Graphite::Colors::Black);
c.fillCircle(400, 300, 100, Graphite::Colors::Red);
c.saveToPNG("out.png");

◆ randomUint32()

uint32_t randomUint32 ( )
inline

Returns a uniformly-distributed random 32-bit unsigned integer.

Uses a thread-local Mersenne-Twister seeded from std::random_device. The generator and distribution are static, so the first call pays the seed cost; subsequent calls are cheap.

Returns
A random value in [0, UINT32_MAX].
